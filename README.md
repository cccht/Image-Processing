# Image-Processing

#### 简介

Image-Processing为图像处理小作业，使用 `pyqt5` 以及 `opencv_python` 编写

其中需要安装的模块：

```python
opencv_python==4.5.3.56
numpy==1.21.2
PyQt5==5.15.4
```

#### 预期功能

- [x] 本地图片查看
- [x] 图片位置基本变换
- [x] 亮度、对比度
- [x] 图片的屏幕适应
- [x] 模糊锐化
- [x] 噪声
- [x] 二值
- [x] 饱和度
- [ ] 后续功能待开发

#### 使用

下载 `py` 及 `requirements.txt` 基本安装即可
